1 Python - язык программирования с лаконичным и понятным синтаксисом. Пирменяется во многих сферах - от машинного обучения до создания игр. Также на нем пишут программную часть сайтов - бэкенд. Бэкенд Dropbox, Netflix, Spotyfy, Youtube частично или полностью написан на Pyton.
Django - один из самых популярных фреймворков для веб-разработки на Python. С помощью него можно создавать сайты любых масштабов и жанров. Например, компания Google использует Django для своего гланого проекта - поисковика.
BASH - програмная оболочка, которая позволяет взаимодейсвовать с программами и сисемами компьютера с помощью команд.
Nginx - веб сервер, который может обрабатывать десяткки тысяч запросов в секунду. Треть всех сайтов в интернете используют nginx. Этот веб-сервер разработали два программиста из России: Игорь Сысоев и Масим Коновалов.
GitHub - сервис для хранения, управления и совместной разработки кода. Пос своей сути- это социальная сеть для разработчиков, которая дает возможность реализовывать свои идеи и свободно обмениваться информацией.
Gunicorn - wsgi-сервер, который обеспечивает взаимодействие между python-фреймворком и веб-сервером, например, между Django и Nginx.
Git - система контроля версий. Используется для хранения кода, помагает ууправлять версиями проекта и деалет удобной коммандную разработку. Git - ключевой инструмент в работе любого программиста.
Docker - программа, которая позволяет работать с приложениями в изолированной среде - в контейнерах. В конетйнере может храниться операционная система, файлы программы, веб-сервер и другие нужные вещи для работы приложения. По сути, это возможность передать свое приложение другому разработчику "вместе с компьютером".
PostgreSQL - система управления базами данных(СУБД) с открытым исходным кодом. Поддерживает большиство современных функций для работы с базами данных.
SQL - специальный стурктурированный язык запросов для работы с базами данных.
